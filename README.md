# 提问箱更新日志

记录每一次的变化 🎉

---

### 2026-06-26

#### ✨ 新功能
- **点赞**：每条公开提问下可以点赞啦，每人限点一次，可取消
- **精选提问 ⭐**：管理员精选的问题会在卡片上显示星星标记
- **日期检索 📅**：按日期筛选提问，快速找到想看的内容
- **卡片开放 🎴**：任何人都可以为自己喜欢的问答生成分享卡片

#### 🔧 优化
- 时间显示更清晰：超过一天的提问直接显示日期，不再只显示「昨天」
